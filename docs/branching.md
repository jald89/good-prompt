# Estrategia de ramas

Se creó la rama `feature/promptscore-clone` para agrupar todas las funcionalidades del clon de PromptScore. Usa esta rama como base para cualquier ajuste adicional relacionado con la UI, backend y documentación generados hasta ahora.

```bash
git checkout feature/promptscore-clone
```

Recuerda abrir Pull Requests contra esta rama cuando añadas mejoras al clon.
